# Standalone CQELS Engine in Java
This under-development version is the next generation of [ prototyped CQELS engine released in 2013](https://github.com/cqels/CQELS-1.x), it will be released by June, 2018 with following features:

1. Support recommended Syntaxes of [RSP Community Group] (https://www.w3.org/community/rsp/)
2. Support CEP operators
3. Support RDFS/OWL reasoning
4. Support Websocket
5. Support Persistent stream
